# ATM Simulation System (Core Java)

## Description
A console-based ATM Simulation System developed using Java, designed to demonstrate basic banking operations such as balance checking, withdrawal, deposit, PIN authentication, and mini statement generation.
This project is suitable for academic mini-projects, Java practice, and viva demonstrations.
